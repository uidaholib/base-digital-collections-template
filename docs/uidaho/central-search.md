# Central Search Set up

Our centralized search index is based on base Dublin Core elements:

contributor, coverage, creator, date, description, format, identifier, language, publisher, relation, rights, source, subject, title, type

Each collection's custom template fields will be mapped to the appropriate core fields using "config-search-index.csv"
This will output a "search-index.json" in the data folder which will be used by the central index to access the metadata.
